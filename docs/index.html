Features

Built to leave no trace.

A clean break, every single time. Nothing remembers you — not the browser, not the web, not the disk.

Fresh identity each launch

A brand-new profile every time you open it. No cookies, cache, storage or logins ever carry over.

Visible wipe on exit

A real terminal opens and runs every deletion command. You watch your tracks burn — no hidden magic.

Randomized fingerprint

A fresh, realistic user-agent is generated each session, so no two visits look like the same machine.

Every site just works

Full Chromium engine under the hood. YouTube, web apps and the modern web all run, exactly as expected.

Kill-session button

One click nukes the current identity, any time you want. Burn it down and start clean instantly.

Nothing phones home

No accounts, no telemetry, nothing saved or sent. Fully open source — read every line yourself.

How it works

Launch · Browse · Burn

Three moments. The last one happens in the open, right in front of you.

01

Launch

Open ITERA and a fresh identity is minted on the spot — clean profile, new fingerprint.

02

Browse

Use the full web freely. Everything lives only for this session, nowhere permanent.

03

Burn

Close it or hit Kill session. A terminal opens and erases every trace from disk.
